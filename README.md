# CTDash

A real-time CT department management simulation game written in pure Python.

Manage patient flow through a radiology CT department across a simulated work shift. Balance scanner throughput, transport logistics, contrast timing, and staffing to maximise your score.

## Quick Start

```bash
python -m CTDash.main
```

## Run Tests

```bash
python -m unittest discover -s CTDash/tests
```

## Requirements

Python 3.11+ — standard library only, no external dependencies.
